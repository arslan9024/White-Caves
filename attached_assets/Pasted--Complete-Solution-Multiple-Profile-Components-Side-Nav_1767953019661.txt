# **Complete Solution: Multiple Profile Components & Side Navigation Bar Issues**

## **üîç Root Cause Analysis**

### **Problem 1: Multiple Profile Components**
```
CURRENT ISSUES:
1. Three separate profile components exist:
   - Navbar Profile (top-right, minimal)
   - Sidebar Profile (left, expanded)
   - Dashboard Profile (middle, different styling)
   
2. Each has:
   - Different HTML structure
   - Different CSS styling
   - Different data fetching logic
   - Different interaction patterns

3. Results in:
   - Visual inconsistency
   - Code duplication
   - Maintenance nightmare
   - Confusing user experience
```

### **Problem 2: Side Navigation Bar Issues**
```
CURRENT ISSUES:
1. Design inconsistencies:
   - Inconsistent spacing
   - Poor typography
   - Bad color choices
   - Unresponsive behavior

2. Functionality problems:
   - Broken navigation links
   - Incomplete assistant listing
   - No collapse/expand functionality
   - Poor mobile experience

3. Performance issues:
   - Unoptimized rendering
   - No lazy loading
   - Excessive re-renders
```

// File: /src/contexts/ProfileContext.jsx
import React, { createContext, useState, useContext, useEffect } from 'react';

const ProfileContext = createContext();

export const ProfileProvider = ({ children }) => {
  const [userProfile, setUserProfile] = useState(null);
  const [notifications, setNotifications] = useState([]);
  const [loading, setLoading] = useState(true);
  
  // Single API call for ALL profile data
  useEffect(() => {
    const fetchProfileData = async () => {
      try {
        // Fetch all profile-related data once
        const [profileRes, notificationsRes] = await Promise.all([
          fetch('/api/user/profile'),
          fetch('/api/user/notifications')
        ]);
        
        const profileData = await profileRes.json();
        const notificationsData = await notificationsRes.json();
        
        setUserProfile(profileData);
        setNotifications(notificationsData);
      } catch (error) {
        console.error('Failed to fetch profile data:', error);
      } finally {
        setLoading(false);
      }
    };
    
    fetchProfileData();
  }, []);
  
  // Update function for all profile instances
  const updateProfile = (updates) => {
    setUserProfile(prev => ({ ...prev, ...updates }));
  };
  
  const value = {
    userProfile,
    notifications,
    loading,
    updateProfile,
    markNotificationAsRead: (id) => {
      setNotifications(prev => prev.filter(n => n.id !== id));
    }
  };
  
  return (
    <ProfileContext.Provider value={value}>
      {children}
    </ProfileContext.Provider>
  );
};

// Custom hook for easy access
export const useProfile = () => {
  const context = useContext(ProfileContext);
  if (!context) {
    throw new Error('useProfile must be used within ProfileProvider');
  }
  return context;
};

#### **Step 2.2: Create Navigation Context**

// File: /src/contexts/NavigationContext.jsx
import React, { createContext, useState, useContext } from 'react';

const NavigationContext = createContext();

export const NavigationProvider = ({ children }) => {
  const [sidebarCollapsed, setSidebarCollapsed] = useState(false);
  const [activePath, setActivePath] = useState('/dashboard');
  const [navigationHistory, setNavigationHistory] = useState(['/dashboard']);
  
  const toggleSidebar = () => {
    setSidebarCollapsed(prev => !prev);
  };
  
  const navigate = (path) => {
    setActivePath(path);
    setNavigationHistory(prev => {
      const newHistory = [...prev];
      // Keep only last 10 navigation items
      if (newHistory.length >= 10) {
        newHistory.shift();
      }
      newHistory.push(path);
      return newHistory;
    });
  };
  
  const goBack = () => {
    if (navigationHistory.length > 1) {
      const previousPath = navigationHistory[navigationHistory.length - 2];
      setActivePath(previousPath);
      setNavigationHistory(prev => prev.slice(0, -1));
    }
  };
  
  const value = {
    sidebarCollapsed,
    activePath,
    navigationHistory,
    toggleSidebar,
    navigate,
    goBack
  };
  
  return (
    <NavigationContext.Provider value={value}>
      {children}
    </NavigationContext.Provider>
  );
};

export const useNavigation = () => {
  const context = useContext(NavigationContext);
  if (!context) {
    throw new Error('useNavigation must be used within NavigationProvider');
  }
  return context;
};
